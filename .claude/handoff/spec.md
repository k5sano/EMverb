\\n\\n# EMVerb プラグイン仕様書\\n\\n## 概要\\n\\nEarthQuaker Devices的な空間系サウンドを目指したDattorroプレートリバーブVST3/AUプラグイン。\\nMutable Instruments CloudsのDSPトポロジーをベースに、EQD的な質感（粒感・揺らぎ・フィルターの色付け）を付加する。\\n\\n## ターゲット環境\\n\\n- フォーマット: VST3 / AU\\n- チャンネル: Stereo in / Stereo out\\n- サンプルレート: 44100 / 48000 / 88200 / 96000 Hz 対応\\n- ビット深度: 32-bit float\\n- バッファサイズ: 32〜2048 samples\\n\\n## パラメータ仕様\\n\\n| ID | 表示名 | 範囲 | デフォルト | 単位 | 説明 |\\n|---|---|---|---|---|---|\\n| \`decay\` | Decay | 0.0〜1.0 | 0.5 | - | リバーブの減衰時間。タンクループのフィードバック量に対応 |\\n| \`damping\` | Damping | 0.0〜1.0 | 0.7 | - | タンク内LPFのカットオフ。高域の減衰速度を制御 |\\n| \`diffusion\` | Diffusion | 0.0〜1.0 | 0.625 | - | 入力拡散段とタンク内APFの係数。音の密度感を制御 |\\n| \`amount\` | Mix | 0.0〜1.0 | 0.5 | - | Wet/Dryミックス比 |\\n| \`input\_gain\` | Input Gain | -18.0〜+6.0 | 0.0 | dB | 入力ゲイン |\\n| \`mod\_speed\` | Mod Speed | 0.0〜1.0 | 0.5 | - | LFO速度。タンク内ディレイ変調の速度を制御 |\\n\\n## サウンドデザイン目標\\n\\n- EarthQuaker Devices（Afterneath / Spectre）的な「粒感」と「揺らぎ」\\n- デジタルアーティファクトを音楽的に聴かせるサチュレーション\\n- アナログフィルター的な色付けのある高域減衰\\n- 左右非対称なモジュレーションによる立体感\\n\\n## 将来追加予定パラメータ（未実装）\\n\\n- \`shimmer\` : ピッチシフトフィードバック（+1oct）\\n- \`filter\_freq\` : タンク後段のSVFカットオフ\\n- \`filter\_res\` : SVFレゾナンス\\n- \`env\_mod\` : エンベロープフォロワー連動のフィルターモジュレーション\\n\\n
